Full-featured editor

CKEditor 5 is a highly flexible framework that lets you build your custom editor of any type (like classic, inline, distraction-free, or document-like), with any set of features and the toolbar type that you need in no time.

This custom editor build contains almost all non-collaborative CKEditor 5 features. You can use it to create your own content or to paste some existing content from Microsoft Word, Google Docs, text documents or any online resources.

This editor was configured specifically to allow testing as many features as possible in one demo, with a multiline toolbar that gives you easy access to all available features. It is based on classic editor, providing you with a boxed editing area with a toolbar, placed in a specific position on the page. Thanks to the autoformatting feature you can also use Markdown-like inline shortcodes as you type to create and format your content without using the toolbar buttons.

Check out the demo of a collaborative editor, too, to try out features such as comments, track changes and real-time collaboration.

Handheld emperor

Nintendo, a Japanese electronics company, started as a hanafuda cards manufacturer in 1889. In the mid-1970s, they entered the early video games market. Nintendo has produced arcade machines, video games, and — since 1985 — home video game consoles. Although it introduced consoles like Nintendo Entertainment System, Super Nintendo Entertainment System, or the Wii, the Japanese gaming giant will always be most cherished for its handheld gaming devices, especially the Game Boy.

A countdown of Nintendo handhelds

  1. Game & Watch was Nintendo's first product offering out-of-home gaming. These small handhelds featured a preprinted LCD screen with black-and-white action displayed, following the board printed on-screen. From 1980 to 1991, over a hundred games were released, gaining great popularity.
  2. In 1989,  the original Game Boy was released. The new amazing machine utilized a cartridge system, allowing the users to play a multitude of games of all kinds. This was a historical game-changer. The first console was soon followed by more developed versions: Game Boy Color and Game Boy Advance. 
  3. In 2004, Nintendo introduced a new console family called Nintendo DS. It sported a dual LCD screen in a folded shell, with the lower one being a touchscreen. Several versions were developed, including Nintendo 2DS and the famous Nintendo 3DS, which offered a stereoscopic 3D display.
  4. 2017 brought the hybrid experience for both couch-preferring gamers and handheld enthusiasts with the release of the Nintendo Switch. It offers both a TV mode with high-definition graphics and a handheld mode using the built-in 6.2‚Äù display, bringing in the best from both gaming worlds. 

Handheld consoles' popularity

While the most recent Switch is a prevalent choice nowadays, the 2DS and 3DS consoles are still popular. The king, however, is none other than the original wonder — the Game Boy.
